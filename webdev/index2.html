Headings in HTML

In HTML, headings are created using the tags <h1> to <h6>. These tags define headings from the most important (h1) to the least important (h6).

Example:

This is Heading 2

This is Heading 3

This is Heading 4

Text Formatting in HTML

HTML provides tags for formatting text:

  • <b> - Makes text bold. Example: Bold Text
  • <i> - Makes text italic. Example: Italic Text
  • <u> - Underlines text. Example: Underlined Text

Subscript and Superscript

The <sub> and <sup> tags are used for subscript and superscript text, respectively.

Examples:

Chemical Formula: CO2

Mathematical Rank: 5th place

Images in HTML

Images are added to a webpage using the <img> tag. Important attributes include src (source), alt (alternative text), and width (image width).

Syntax:

<img src="image.jpg" alt="description" width="400">

Videos in HTML

Videos can be embedded using the <video> tag. Common attributes include controls (play, pause, volume), autoplay, muted, loop, and preload.
